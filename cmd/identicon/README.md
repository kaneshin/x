# Identicon

Generating identicon image!

![kaneshin.png](https://github.com/kaneshin/identicon/raw/master/kaneshin.png)

## Installation

```shell
$ go get -u github.com/kaneshin/x/cmd/identicon
```

## Usage

```shell
$ echo kaneshin | identicon > kaneshin.png
```
